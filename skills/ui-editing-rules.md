# Skill: UI Editing Rules

Use this skill whenever changing the interface.

## Preserve the feel

The app should stay:

- Soft
- Clean
- Warm
- Personal
- Simple
- Mobile-friendly

## Preserve structure

Do not remove:

- Weekly cards
- Collapsible fixed bills
- Separate cash-flow tab
- Locked cash-flow constants
- Finalize/unfinalize week controls

## Avoid clutter

Do not show all configuration rows at once.

Prefer collapsible sections and tabs.

## Primary hierarchy

Most important:

1. Weekly allowance
2. Spent
3. Left over
4. POT after week

POT should be visible but not overemphasized.
