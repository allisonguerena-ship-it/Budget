# Weekly Card Example

A weekly card should roughly contain:

```txt
Week 1: Jul 1 – Jul 7
Status: Open or Finalized

Fixed bills used: $3,504.49
Allowance: $507.27
Spent: $430.12
Left over: $77.15
POT after week: $77.15

Expense rows:
- Date
- Cost / Purchase
- Category
- Amount

Buttons:
- Add cost
- Finalize week

If finalized:
- Inputs disabled
- Badge says Finalized
- Button says Unfinalize week
```
