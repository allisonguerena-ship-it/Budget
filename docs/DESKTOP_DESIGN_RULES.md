# Desktop Design Rules

Desktop should use available space without becoming cluttered.

Do not simply stretch the mobile layout.

## Layout rules

Desktop may use:

- Side-by-side sections
- Wider tables
- Multi-column cards
- Persistent summary areas

Desktop should still feel simple and calm.

## Editing rules

Desktop should support faster editing than mobile.

Use:

- Keyboard-friendly fields
- Clear tab order
- Easy row editing
- Visible summaries near the top

## Avoid

Avoid:

- Overcrowded dashboards
- Too many panels
- Too many filters
- Huge permanently expanded sections
- Spreadsheet-like density unless explicitly needed

## Desktop testing checklist

After UI changes, test:

- 13-inch laptop width
- Full desktop width
- Tables with many rows
- Week navigation
- Fixed bill editing
- Cash-flow planner
